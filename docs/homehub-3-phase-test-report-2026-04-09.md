## 3. 阶段 1：正常语言回答

### 用例 1.1

- 输入：`你好。`
- 预期：
  - 走本地普通问候链
  - 返回类似“你好，有什么可以帮忙的”
  - 不误触发创建智能体
  - 不误触发联网查询

结果：通过  
说明：上一轮已验证普通问候不会再误走智能体工厂或联网兜底。

### 用例 1.2

- 输入：`福冈今天的天气怎么样，温度多少。`
- 预期：
  - 优先走天气专用链
  - 能返回地点、天气情况、温度
  - 地点解析失败时进入联网天气兜底，而不是直接回默认 weather state

结果：通过  
说明：上一轮已验证天气查询链可正常返回天气信息，并修复了默认天气回退和错误链接直出问题。

### 用例 1.3

- 输入：`我要从日本去美国，帮我查询5月31号的所有去美国的飞机，东京到旧金山的时间和价格`

  - 自动联网搜索

结果：通过  
说明：上一轮验证的是 HomeHub 的联网搜索与总结链路本身，不是特定商业机票源的精确实时抓取。
