### 具体问题发现

#### 问题1️⃣: 问候识别误触发
```
查询: "你好啊 HomeHub"
预期: 返回自然问候
实际: "还有其他要求吗？如果没有，回答'确认创建'。"
```
**根本原因**: NLU将带有"HomeHub"的特定问候误识别为智能体创建确认
**影响**: 1/4 问候测试失败 (25% 失败率)

#### 问题2️⃣: 文件搜索路径访问失败
```
查询: "搜索 /tmp/homehub-family-suite/family-inbox 下面的 budget 文件"
预期: 搜索或降级
实际: "搜索目录不可用：path_not_found"
```
**根本原因**: 搜索模块路径解析问题
**影响**: 2/12 文件操作测试失败 (17% 失败率)
